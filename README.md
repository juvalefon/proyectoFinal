## La proliferación de excremento de perro en el parque público Alcázares, ubicado en Bogotá-Colombia
### Autor: juvfonsecar@udistrital.edu.co

Este proyecto surge como una respuesta consciente ante la constante presencia de excrementos caninos en el Parque Alcázares de Bogotá, una situación que afecta la salubridad, el entorno y el bienestar de quienes lo disfrutan.
