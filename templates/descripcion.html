Tecnologías Utilizadas

Para el desarrollo de este proyecto, se están empleando y se planea utilizar un conjunto de tecnologías web modernas y bibliotecas especializadas.

HTML5

Es el esqueleto de todas las páginas, estructurando el contenido de manera semántica para asegurar que la información sea accesible y bien organizada.

CSS3

Se encarga de toda la presentación visual. Mediante hojas de estilo generales y específicas, se ofrece una experiencia de usuario agradable e intuitiva.

JavaScript

Aporta la interactividad y dinamismo a las páginas, crucialmente para interactuar con bibliotecas de mapeo y visualización de datos.

Leaflet.js

Biblioteca ligera y potente para crear los mapas interactivos, visualizar el Parque Alcázares y representar los polígonos geográficos.

Turf.js

Se planea integrar esta biblioteca para análisis geoespacial avanzado, permitiendo realizar operaciones como análisis de densidad y cálculos de áreas directamente en el navegador.

La combinación de estas tecnologías nos permite construir una aplicación web que no solo presenta información sobre la problemática, sino que también busca ofrecer herramientas para la sensibilización, el análisis y la visualización interactiva de datos.

La metodología combina una fase descriptiva-diagnóstica con una de desarrollo tecnológico para crear una plataforma web interactiva que sirva como herramienta para la concienciación comunitaria.
