Herramientas y Enfoque

El presente proyecto se enfoca en generar conciencia sobre la problemática ambiental derivada de los excrementos caninos en el Parque Alcázares, en Bogotá. El objetivo es informar, sensibilizar y promover el cuidado del espacio público a través de una experiencia digital accesible, visualmente atractiva e interactiva.

Para ello, se emplean herramientas web como HTML, CSS, JavaScript y Leaflet.js. Esta combinación tecnológica permite construir una solución informativa y visual que facilita la comprensión del problema y fomenta un comportamiento más responsable con el entorno.

Tecnologías Utilizadas

Para el desarrollo de este proyecto se emplean las siguientes tecnologías:

HTML

Estructura el contenido de forma clara y semántica.

CSS

Permite una presentación visual intuitiva y agradable.

JavaScript

Otorga dinamismo e interactividad a la plataforma.

Leaflet.js

Biblioteca para crear mapas interactivos y ligeros.

Turf.js

Se planea integrar para análisis geoespacial avanzado.
